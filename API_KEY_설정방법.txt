===== ANTHROPIC_API_KEY 설정 방법 =====

AI 글 자동 생성 기능을 사용하려면 Anthropic API 키가 필요합니다.

[API 키 발급]
1. https://console.anthropic.com 접속
2. 회원가입 / 로그인
3. API Keys 메뉴에서 "Create Key" 클릭
4. 생성된 키 복사 (sk-ant-... 형태)

[API 키 설정 방법 1 - 영구 설정 (권장)]
1. Windows 시작 버튼 우클릭 → "시스템" 클릭
2. "고급 시스템 설정" → "환경 변수"
3. "시스템 변수" 에서 "새로 만들기" 클릭
4. 변수 이름: ANTHROPIC_API_KEY
5. 변수 값: sk-ant-xxxxxx (발급받은 키 입력)
6. 확인 후 컴퓨터 재시작

[API 키 설정 방법 2 - 임시 설정]
시작.bat 를 수정해서 맨 위에 아래 줄 추가:
  set ANTHROPIC_API_KEY=sk-ant-여기에키입력

[요금 안내]
- Claude claude-haiku-4-5 모델 사용 (가장 저렴한 모델)
- 글 1편 생성 시 약 $0.008 (약 11원) 수준
- 무료 크레딧 $5로 약 600편 생성 가능
- 무료 크레딧으로 시작 가능
